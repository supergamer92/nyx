# Nyx OS Branding Assets

This directory contains all visual identity assets for Nyx OS.

## Structure

```
nyx-branding/
├── wallpapers/          # Default wallpaper collection
│   ├── default-dark.jpg
│   ├── default-light.jpg
│   └── ...
├── icons/               # Nyx icon set (SVG)
│   ├── apps/
│   ├── actions/
│   ├── status/
│   └── devices/
├── sounds/              # System sounds
│   ├── login.ogg
│   ├── notification.ogg
│   ├── screenshot.ogg
│   └── volume-change.ogg
├── fonts/               # Bundled fonts
│   ├── Inter/
│   └── JetBrainsMono/
├── boot-splash/         # Plymouth boot theme
│   └── nyx/
├── cursor/              # Custom cursor theme
│   └── nyx-cursor/
└── logo/                # Nyx OS logos
    ├── nyx-logo.svg
    ├── nyx-wordmark.svg
    └── nyx-icon.svg
```

## Design Tokens

- **Primary color**: `#7c5cfc` (Indigo-purple)
- **Dark background**: `#0a0a0c`
- **Light background**: `#fafaf9`
- **Font UI**: Inter (variable weight)
- **Font Code**: JetBrains Mono
